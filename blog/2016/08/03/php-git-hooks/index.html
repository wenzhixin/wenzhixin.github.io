如何简单的在 Php 下运行 Git Hook，实现自动部署

Aug 3rd, 2016 10:25 am

由于 php 是运行在 nginx 下的，直接使用 shell_exec 会出现权限问题，所以我们首先需要解决权限问题。

  1. 登录到 www-data 用户
  1. 创建 id_rsa 并将内容复制到目标 git 中
  1. 测试并 clone 所需要的仓库
  1. 添加 api.php 文件
  1. 在 git 仓库中设置 WebHooks 即可
